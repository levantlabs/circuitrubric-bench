* Output Branch: M3 (Out), M4 (Cascode Out 1), ... M9 (Cascode Out 8)
* Note: To achieve a 1-to-8 ratio, the reference current must be set by an external source.
* The output branch consists of 8 parallel cascoded pairs to multiply the current by 8.
